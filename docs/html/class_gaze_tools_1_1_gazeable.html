Detailed Description

Provides an interface for "registering" gazers, and privately managing its Basically a very fancy wrapper for a single boolean value; Gazed At yes/no. "gazed-at" state is based on if there's at least one registered active gazer. Invokes Gaze Start/Change/End events.
